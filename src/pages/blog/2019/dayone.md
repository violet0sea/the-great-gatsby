使用 Gatsby 构建博客，Gatsby 是一个基于 React 和 Graphql 的静态 html 构建网站，可以用于博客搭建，使用 markdown 文件，通过 markdownremark 转化为 html

```
$ yarn add gatsby -g
```

```
$ gatsby new <project-name>
```

```
$ gatsby develop
```

or use yarn start | yarn develop

打开http://localhost:8000
